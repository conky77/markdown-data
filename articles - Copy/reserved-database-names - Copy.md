# Reserved Database Names
A reserved database name is a name that cannot be assigned to a user database in SQL Server PDW.  
  
List of reserved database names:  
  
-   master  
  
-   model  
  
-   msdb  
  
-   tempdb  
  
-   pdwtempdb  
  
-   pdwtempdb1  
  
-   pdwtempdb2  
  
-   DWConfiguration  
  
-   DWDiagnostics  
  
-   DWQueue
